# Автотесты для сервиса nbank

Этот проект содержит автотесты, покрывающие функционал:
- Депозит денег пользователем
- Перевод денег с одного счета на другой
- Изменение имени пользователя

# Чек-лист проверок для API:
**Депозит денег пользователем:**
- Пополнение баланса (позитивные сценарии):
  - Пополнение с целой суммой
  - Пополнение с 1 знаком после запятой
  - Пополнение с 2 знакакми после запятой
  - Пополнение на минимальную сумму
  - Пополнение на максимальную сумму
- Невалидные значения суммы:
  - Пополнение на нулевую сумму
  - Пополнение на сумму, превышающую максимальную
  - Пополнение на отрицательную сумму
- Пополнение чужого баланса
- Пополнение баланса на несуществующем id счета
- Пополнение баланса от имени администратора
- Пополнение баланса неавторизованным пользователем
- Получение информации о депозите

**Перевод денег с одного счета на другой:**
- Перевод между своими счетами:
  - Перевод целой суммы
  - Перевод суммы с 1 знаком после запятой
  - Перевод суммы с 2 знаками после запятой
  - Перевод минимальной суммы
  - Перевод максимальной суммы
- Перевод на счет другого пользователя:
    - Перевод целой суммы
    - Перевод суммы с 1 знаком после запятой
    - Перевод суммы с 2 знаками после запятой
    - Перевод минимальной суммы
    - Перевод максимальной суммы
- Перевод невалидной суммы
  - Перевод суммы, превышающей максимальную
  - Перевод нулевой суммы
  - Перевод отрицательной суммы
- Перевод с чужого счета
- Перевод с несуществующего счета
- Перевод на несуществующий счет
- Перевод на тот же самый счет
- Перевод от имени администратора
- Перевод неавторизованным пользователем
- Получение информации о транзакциях

**Изменение имени пользователя:**
- Изменение имени (позитивные сценарии):
  - Изменение имени с верхним регистром
  - Изменение имени с нижним регистром
  - Изменение имени со смешанным регистром
- Изменение имени (некорректные имена):
  - Имя из одного слова
  - Имя из трех слов
  - Имя с двумя пробелами
  - Имя, содержащее цифру
  - Имя, содержащее "_"
  - Имя, содержащее "-"
  - Имя, содержащее "."
  - Имя на кириллице
  - Имя состоящее из пробелов
  - Пустое имя

# Тест-кейсы для UI:
**Депозит денег пользователем:**

✅ Название: Успешное пополнение баланса  
⚠️ Предусловия:  
1 Пользователь авторизован и имеет счет  
2 Открыта страница "User Dashboard"  
➡️ Шаги:

| # | Шаг                                                        | Ожидаемый результат                                |
|---|------------------------------------------------------------|----------------------------------------------------|
| 1 | Нажать кнопку "Deposit Money"                              | Открылась страница "Deposit Money"                 |
| 2 | В дропдауне "Select Account" выбрать счет                  | Счет выбран                                        |
| 3 | В поле ввода "Enter Amount" ввести валидное значение суммы | Сумма введена                                      |
| 4 | Нажать кнопку "Deposit"                                    | Появляется алерт с сообщением об успешном депозите |
| 5 | Нажать кнопку "OK" на алерте                               | Выполняется переход на страницу "User Dashboard"   |
| 6 | Проверить в API, что баланс был пополнен                   | Значение баланса увеличилось на сумму депозита     |
<br>

✅ Название: Пополнение баланса на сумму, превышающую максимальную
